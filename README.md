#  PT_INTI_APP

Aplikasi **PT_INTI_APP** merupakan aplikasi berbasis mobile yang dikembangkan untuk mempermudah aktivitas karyawan di lingkungan perusahaan, seperti **Login**, **Absensi**, dan **Employee Self Service (ESS)**.

Aplikasi ini dirancang agar dapat digunakan kapan saja dan di mana saja, dengan memperhatikan aspek keamanan sehingga membantu mengurangi risiko serangan siber terhadap sistem perusahaan.

# 🚀 Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/username/repository.git
```

### 2. Masuk ke Folder Project

```bash
cd repository
```

### 3. Install Dependency Flutter

```bash
flutter pub get
```

### 4. Jalankan Aplikasi

```bash
flutter run
```
